# 🎯 Dashboard de Vendas - Sabores & Aromas

## 📊 Sobre o projecto
Este dashboard foi desenvolvido para a empresa Sabores & Aromas (empresa fictícia), especializada na comercialização de produtos alimentares e bebidas. A análise contempla dados do período de Janeiro a Novembro de 2024, utilizando Excel como fonte de dados e Power BI como ferramenta visualização e análise.

## 🎯 Objectivos
O principal objetivo deste dashboard é fornecer insights sobre o desempenho comercial da empresa, permitindo uma visão dos indicadores financeiros e métricas de vendas.

## 🛠️ Ferramentas Utilizadas
- Microsoft Power BI
- Microsoft Excel

## 📈 Análises Desenvolvidas

O dashboard é composto por diferentes painéis de análise:

### 📊  Indicadores Principais
- Cards com métricas chave: Receita Total, Margem Total e Custo Total

### ⏳ Análise Temporal
- Evolução da Receita ao longo do período (Jan/2024 a Nov/2024)

### 🛒 Análise por Produto
- TOP 5 Grupos de Produtos por Receita, destacando Farinha de Trigo como líder em faturamento
- TOP 5 Grupos de Produtos por Margem, com Cachaça apresentando a maior margem bruta de lucro
- Distribuição de Receita por Linha de Produto (gráfico rosca), com predominância da linha de Alimentos
- Distribuição de Margem por Linha de Produto (gráfico rosca), também liderada pela linha de Alimentos

### 👥 Análise da Equipe Comercial
- Desempenho de Margem por Vendedor e Supervisor

## 💡 Entrega de Valor

### 🎯 Benefícios Directos para a Sabores & Aromas
- **Tomada de Decisão Baseada em Dados**: permite decisões estratégicas fundamentadas em informações concretas sobre vendas e desempenho
- **Optimização de Estoque**: identificação dos produtos mais vendidos, auxiliando no planeamento de compras e gestão de inventário
- **Gestão de Equipe**: monitoramento do desempenho individual dos vendedores e supervisores, facilitando bonificações e treinamentos
- **Controle Financeiro**: visualização clara de receitas, custos e margens, permitindo ajustes rápidos na estratégia comercial
- **Sazonalidade**: Compreensão dos padrões de venda ao longo do ano, auxiliando no planeamento de campanhas e promoções

## 📥 Instruções para Visualizar o Projeto

### ✅ Requisitos
- Microsoft Power BI Desktop instalado (versão mais recente)
- Conta Microsoft para acesso ao Power BI Service (opcional para visualização online)

### 📝 Passos para Visualização
1. **Download do Arquivo**
   - Faça o download do arquivo `.pbix` disponível neste repositório
   - O arquivo está localizado na pasta `dashboard/analise_vendas.pbix`

2. **Abertura do Projecto**
   - Abra o Microsoft Power BI Desktop
   - Clique em "Arquivo" > "Abrir" e selecione o arquivo baixado
   - Aguarde o carregamento completo do dashboard

3. **Navegação**
   - Interaja com os gráficos clicando nos elementos visuais

4. **Alternância entre Temas**
   - Use o botão de alternância de tema no canto superior esquerdo
   - Escolha entre modo claro ou escuro conforme sua preferência

### ⚠️ Observações Importantes
- Para melhor experiência, recomenda-se utilizar a última versão do Power BI Desktop
- Em caso de problemas com conexão de dados, verifique se todas as fontes estão acessíveis
- Para visualização online, é necessário publicar o relatório no Power BI Service
